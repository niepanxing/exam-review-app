一、课堂四要素

  • 内容讲师学员形式
  • 标准课程复制关键 = 课程结构 + 讲师风采 + 演绎呈现 + 资源辅助

以学员为中心四问

  • Who：谁是学员
  • Why：为什么学
  • What：带来什么
  • How：如何学得更好

SUPER原则

  • S - Safe：安全感/被尊重
  • U - Utilization：实用性/解决问题
  • P - Participation：参与学习/承担责任
  • E - Experience：关联实际/体验机会
  • R - Reinforcement：不断强化/反馈

二、讲师七项基本功

  1. 形象
  2. 声音
  3. 眼神
  4. 表情
  5. 姿态
  6. 手势
  7. 位置

声音四要素

  • 音量：视人数教室大小
  • 语调：清晰有变化
  • 语速：120-200字/分钟最佳
  • 停顿：10秒深思 / 6秒略想 / 3秒段落 / 2秒间隔

眼神技巧

  • 必须：接触积极听众、注视几秒再转向、缓慢平均接触、注视关键人物
  • 不能：盯讲稿、看后墙、看天花板、注视某人过长

三、讲授三部曲与开场方法

讲授三部曲

  • 激发兴趣（能量） → 深层互动（转化） → 关闭总结（控场）

开场五种方法

  • 引文法：新闻/典故/名言/视频
  • 提问法：设问/自问
  • 利益分享法：闪光点/价值
  • 故事法：联结主题
  • 数字法：统计数据

四、大脑记忆规律

五大原则

  • 分块原则：7±2
  • 记录原则
  • 联结原则
  • 重复原则：6X
  • 特殊原则：首要原则1 + 最近原则Z

教学方法与刺激度（从低到高）

  1. 阅读理解
  2. 提问发言
  3. 大组讨论
  4. 讲解授课
  5. 小组讨论
  6. 案例分析
  7. 视听教学
  8. 游戏带动
  9. 情景模拟/现场演示

五、互动与演练

万能互动流程

  • Me：个人思考/书写 → We：小组讨论/交流 → Us：全班分享
  • 先封闭再开放

讲授法公式

  • 身份定义：我是XXX + 给XXX用 + 希望XXX + 担心XXX
  • 身份定位：我是XXX + 要为XXX + 解决XXX + 满足XXX

演练法适用场景

  • 行为养成：行为演示 → 行为辨识 → 行为养成 → 鼓励应用
  • 动作技能训练

音乐辅助三个时机

  • ① 快速破冰/控场（Serenade/Bandari）
  • ② 提升课堂氛围（讨论/练习/休息环节）
  • ③ 收官升华（整体总结/学员反思）

自我介绍三要素

  • 姓名：如何让学员记住
  • 个人经历：如何对学员有帮助
  • 我&课程：如何展现专业性
